# Customer Churn Analysis (Excel)

## Objective
Analyze customer churn patterns to identify key factors driving customer attrition and provide insights for retention strategies.

## Dataset
Customer-level dataset containing:
- Customer status (Customer ID, Churn Label, Churn Category, Churn Reason)
- Demographics (Gender, Age, Under 30, Senior)
- Contract information (Contract Type, Payment Method, State, Phone Number, Group, Number of Customers in Group)
- Subscription types and charges

## Excel Techniques Used
- Pivot Tables
- Calculated Fields
- IF functions
- Grouping
- Remove Duplicates
- Conditional Formatting
- Charts and Data Visualization

## Analysis Performed & Key Insights
- **Churn Rate Analysis**  
  Calculated overall churn rate by comparing total customers and churned customers using pivot tables.

- **Churn Reason Analysis**  
  Identified the most common reasons for customer churn using column charts.

- **Competitor-Driven Churn**  
  Analyzed churn caused by competitors using pie charts.

- **Churn by Demographics**  
  Examined churn patterns across gender and age groups.

- **Age Group Analysis**  
  Compared total customers and churn rate across age groups using combo charts.

- **Consumption-Based Churn**  
  Analyzed churn behavior based on data plan type and grouped consumption using stacked column charts.

- **Geographical & Plan Analysis**  
  Evaluated churn rates by state and international plan usage through pivot table analysis.

## Tools
- Microsoft Excel
