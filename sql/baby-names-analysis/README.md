# Baby Names Trend Analysis (SQL)

## Objective
Analyze trends in baby names popularity using SQL.

## Dataset
US Baby Names dataset containing birth year, name, gender, and number of births.

## SQL Concepts Used
- GROUP BY & aggregation
- CASE WHEN
- Window function (RANK)
- Filtering with WHERE
- Ordering and limiting results

## Key Insights
- Identified popular vs non-popular names
- Ranked top male baby names
- Analyzed female name trends across years
